Meus Projetos

Aqui estão alguns dos projetos que desenvolvi e aprendi durante minha jornada como desenvolvedor.

Clonefy

Clonefy é um projeto inspirado no Spotify que eu fiz durante a imersão alura 2025, desenvolvido para praticar React e Tailwind. Ele implementa o consumo de APIs para busca de artistas, proporcionando uma experiência interativa e acessível em diferentes dispositivos.

Desafios: Durante o desenvolvimento, me desafiei a fazer o sistema de pesquisa funcionar em outros dispositivos, além de no ambiente local. Esse requisito não foi ensinado durante a imersão, mas me virei para implementar essa funcionalidade de forma que fosse acessível de qualquer lugar. Além disso, enfrentei desafios na integração com a API de buscas e na organização dos componentes no React

Tecnologias utilizadas: React, HTML, CSS e JavaScript.

Futuramente, o projeto contará com melhorias na interface e uma versão responsiva para diferentes dispositivos.

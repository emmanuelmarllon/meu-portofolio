Meus Projetos

Aqui estão alguns dos projetos que desenvolvi e aprendi durante minha jornada como desenvolvedor.

Spotify

Recriei uma versão simples do Spotify utilizando HTML, CSS e JavaScript. Durante esse projeto, aprendi várias técnicas avançadas de JavaScript, além de aprimorar muito minha habilidade com CSS.

Desafios: Um dos maiores desafios foi implementar a funcionalidade de navegação dinâmica entre as músicas e a criação de uma interface que fosse agradável de usar, com um design que se aproximasse do original do Spotify.

Tecnologias utilizadas: HTML, CSS, JavaScript.

Este projeto foi desenvolvido durante a Imersão Frontend da Alura, onde pude consolidar meus conhecimentos em desenvolvimento web.

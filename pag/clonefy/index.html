Clonefy - Recriação Visual do Spotify

Clonefy é uma recriação visual do Spotify, desenvolvida com HTML, CSS, JavaScript e React. O projeto faz parte de um desafio proposto na Imersão Frontend da Alura, onde o objetivo era criar o front-end da plataforma.

Capturas de Tela

Tecnologias Utilizadas

  • HTML - Estruturação do layout do projeto, com ênfase na responsividade.
  • CSS - Estilização avançada, criando uma interface semelhante ao Spotify.
  • JavaScript - Manipulação do DOM e lógica por trás da interface.
  • React - Aprendizado na criação de componentes reutilizáveis e gerenciamento de estado.
  • API de Busca - Integração de API para pesquisar músicas, artistas e álbuns.

Desafios e Aprendizados

Este projeto foi o meu primeiro contato com o React, e confesso que enfrentei várias dificuldades. Porém, o suporte da comunidade da Alura foi essencial para superar esses obstáculos, especialmente quando eu tinha dúvidas no processo de aprendizado. Além disso, o maior desafio foi lidar com a manipulação do estado no React e integrar isso com o HTML/CSS de forma fluida.

Aprendi bastante sobre a utilização de APIs para fazer a busca de músicas, artistas e álbuns, além de aprimorar minhas habilidades em JavaScript, que eu já tinha algum conhecimento, mas nunca havia aplicado de forma tão prática e profunda. A experiência com React foi marcante, e posso dizer que agora estou muito mais confortável com essa tecnologia.

Melhorias Futuros

Algumas funcionalidades que eu planejo adicionar no futuro incluem:

  • Possibilidade de clicar em um artista e ser redirecionado ao seu perfil no Spotify.
  • Adição de reprodução de músicas diretamente no clone, com controles de play/pause.
  • Versão responsiva para dispositivos móveis, garantindo melhor experiência.

Atenção

A versão mobile ainda não foi implementada neste projeto. Portanto, se você acessar o Clonefy em um dispositivo com resolução menor que 720px, o layout poderá apresentar alguns problemas de exibição. Estamos trabalhando para implementar uma versão responsiva em breve!
